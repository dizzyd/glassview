# GlassView

A client-side mod that makes glass more visible when chiseling in Vintage Story.

## Features

- Renders a semi-transparent highlight overlay on glass voxels when holding a chisel
- Makes it much easier to see and work with glass in chiseled blocks
- Purely client-side - no server installation required

## Building

To build the mod, run:

```bash
./build.sh
```

The built mod will be in the `Releases` directory.

## Installation

Copy the `.zip` file from the `Releases` directory to your Vintage Story `Mods` folder.
